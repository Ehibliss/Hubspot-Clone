# HubSpot Clone

A responsive frontend clone of the HubSpot Webpage built with React. This project focuses on improving UI/UX design skills, component reusability, and real-world frontend development practice.

---

## Features

- Responsive landing page design
- Modern UI inspired by HubSpot
- Reusable React components
- Clean and structured codebase
- Smooth navigation layout
- Mobile-friendly interface

---

## Tech Stack

- React.js
- JavaScript (ES6+)
- CSS3
- HTML5
